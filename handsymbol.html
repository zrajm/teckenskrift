Teckenskrift: Handsymboler

In ASLwrite nomenclature the individual handshape glyphs are called ‘digits’ and the whole collection of them is known as ‘the digibet’.

Even though there is much overlap between the American and the Swedish Sign Languages, both languages have a number of features that are particular to them. The following table is an attempt at creating all the digits necessary for writing Swedish Sign Language (STS).

Different PDF documents found on the ASLwrite website have different versions of the ASLwrite digibet (presumably reflecting different development stages of the writing system). In order to make sense of them I've here tried to document all the differing versions I've found.

I've found Jolanta Lapiak’s handshape charts (some of them listing experimental digits) a good source of inspiration for coming up with the missing digits for STS. (I'm not 100% certain, but these seem to be older ideas, experiments and revisions intended to be used by the ASLwrite and si5s writing systems.)

Fotnoter

[1] Hand Description: Each hand consists of five finger description in the order thumb–index–long–ring–little finger. The following letters are used to indicate finger shape.

  • b = böjt/bent – Finger/thumb angled in same way as v but all joints are slightly bent giving finger/thumb a rounded appearance.
  • f = fyrkant/square – Like k, but innermost joint is not bent.
  • k = knutet/fisted – The finger is maximally bent, in the same way it would be if hand was turned into a fist.
  • r = rak/straight finger – The finger is completely straight (with no bent joints) and finger/thumb is parallel to the palm.
  • v = vinklat/angled – Finger is bent in the innermost joint, but otherwise straight or thumb is angled up from palm.

If the thumb touches other fingers, then thumb + relevant finger cluster is capitalized (finger cluster = some fingers which are in contact with each other). If thumb is annotated as both separated from (’-’) AND touching the index finger then contact is made at the tip only, if touching without separation then there's contact along the full length of the thumb.

The following symbols are used between thumb/fingers:

  • - = spread/separated – These fingers do not touch.
  • + = Crossed (can also come before the thumb to indicate that thumb cross other finger, rather than touch at tip).
  • | = Separated by thumb.
